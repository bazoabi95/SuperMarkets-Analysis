# 🛒 Supermarket Visitor Behavior & Activity Analysis

## 📌 Project Overview
This project analyzes supermarket visitor behavior using geolocation signal data collected from multiple supermarket branches across Israel.

The project combines:
- SQL
- Python
- Regression Analysis
- Data Visualization
- Looker Studio Dashboards

to generate actionable business insights related to:
- customer behavior
- branch load
- operational efficiency
- visitor trends
- supermarket activity patterns

---

# 🎯 Project Objectives
- Analyze supermarket visitor activity patterns
- Detect peak hours and high-load branches
- Identify customer, employee, and supplier behavioral patterns
- Explore relationships between population size and supermarket distribution
- Build regression models to evaluate visitor trends over time
- Generate data-driven business recommendations

# 🔄 Project Workflow

## 1️⃣ Data Extraction & Cleaning
Performed ETL processes using SQL and BigQuery to:
- clean datasets
- transform raw data
- validate signal accuracy
- aggregate activity metrics

---

## 2️⃣ Exploratory Data Analysis (EDA)
Analyzed:
- supermarket distribution by city
- population vs supermarket ratios
- visitor activity trends
- operational load patterns
- customer traffic behavior

Visualizations included:
- heatmaps
- scatter plots
- correlation analysis
- activity trend charts

---

## 3️⃣ Behavioral Segmentation
Developed logic-based behavioral classifications to identify:
- Employees
- Suppliers
- Regular Customers

Segmentation was based on:
- visit frequency
- activity duration
- active hours
- signal accuracy
- branch activity patterns

---

## 4️⃣ Regression Analysis
Built regression models to analyze:
- visitor trends over time
- branch activity changes
- store load patterns
- area-to-visitor ratios

---

## 5️⃣ Dashboard Development
Built interactive dashboards using Looker Studio to visualize:
- hourly supermarket activity
- weekly traffic trends
- branch load patterns
- operational KPIs
- customer activity behavior

---

# 📊 Key Insights
- Fridays and Thursdays showed the highest supermarket activity
- Some branches experienced significantly higher operational load
- Visitor patterns varied between cities and supermarket branches
- Certain branches showed declining visitor trends over time
- Population size alone did not fully explain supermarket density
- Signal analysis enabled identification of employee and supplier behavior

---

# 💡 Business Recommendations
- Optimize staffing during peak activity hours
- Improve operational planning using visitor trends
- Reduce customer congestion during busy hours
- Use behavioral segmentation for operational optimization
- Apply regression insights for forecasting and decision-making
